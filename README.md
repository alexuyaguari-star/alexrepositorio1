# valor agregado bruto por cantones del Ecuador Dataset

Este repositorio contiene datos públicos del VAB Cantonal. El archivo `enemdu_2025.csv` incluye variables como agricultura, comercio, servicios profesionales etc.

## Fuente
Datos del banco central del ecuador.

## Uso
Este dataset puede usarse para visualización geoespacial, análisis regionales, y entrenamiento de modelos predictivos.
